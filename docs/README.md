```
証明の柱：
・天秤の偽装メカニズム（悪が善に偽装できる構造的欠陥）
・物理法則による悪の中心への引き寄せ
・二進数の全パターン（00・01・10・11）を三者では網羅できない
・日本の四柱の神との対応
```

**→ 社会制度・政治・法律に関心のある人はここから読む**

## docs/ と foundations/ の関係

```
foundations/（根拠群）          docs/（実践群）

「なぜ正しいか」               「何を主張し、どう使うか」
数学的・物理的証明              理論定義・手順・応用事例

　　　└──────────── 相互参照 ────────────┘
　　　　　　　　　　　　↓
　　　　　　　TTT理論の完全な体系
```

`foundations/` の証明が `docs/` の主張を支え、`docs/` の応用事例が `foundations/` の意義を示す。二つのフォルダは相互補完の関係にある。
