## 検証結果 2: 外接球/内接球半径比と双対ペア

## 検証結果 3: 黄金比 φ の出現境界

- **正二十面体**: 頂点座標の非零成分の大きさ比 = $\varphi = \frac{1+\sqrt 5}{2}$（厳密一致）。$\varphi^2 - \varphi - 1 = 0$ 確認
- **正十二面体**: 座標成分の大きさが厳密に等比数列 $1/\varphi : 1 : \varphi$（公比 φ）
- **正四面体・正八面体・正六面体**: 全座標が**有理数**（φ を含まない）

**機械検証された事実**: 五体は「φ-free 群（4, 6, 8点）」と「φ 群（12, 20点）」に厳密に二分される。

**理論的含意 [C]**: 対数螺旋と黄金比の内在的関係を踏まえると、螺旋展開の「前半（有理座標系）」と「後半（φ 座標系）」で生成規則が質的に転換する可能性。Seven Stars 9 Code の二十面体 3×4 構造との接続点。
