# 機械検証レポート:D6 ⊂ E8 ルート系包含

**検証スクリプト:** `verify_D6_in_E8.py`(Python 3 + sympy — 厳密有理数演算)
**実行日:** 2026年6月11日
**結果:** 全項目パス — 分類 [A](数値的・機械的に検証済み)

## 検証した命題

D6型ルート系は、E8型ルート系の中に**閉部分系(closed subsystem)**として埋め込まれる。具体的には、E8の240本のルートのうち60本がD6ルート系を成し、その単純ルートのCartan行列はD6型標準行列と厳密に一致する。

## 検証方法

すべての計算は `sympy.Rational` による厳密有理数演算で行った。浮動小数点近似を一切含まないため、結果は「数値的に近い」という近似的主張ではなく、記号的・厳密な事実の確認である。

1. **E8の構築。** ℝ⁸内の標準実現でE8の240ルートを構築:整数型ルート ±eᵢ±eⱼ(i<j)が112本、半整数型ルート(±½, …, ±½)でマイナス符号が偶数個のものが128本。本数 = 240、全ルートのノルム² = 2 を確認。

2. **D6の同定。** E8ルートのうち、最初の6座標のみに台を持ち成分がすべて整数であるものを抽出。ちょうど60本の ±eᵢ±eⱼ(0 ≤ i < j ≤ 5)型ルートが得られ、これはD6の標準実現に一致する。

3. **Cartan行列の検証。** 単純ルートとして α₁…α₅ = eₖ−eₖ₊₁(k=0..4)、α₆ = e₄+e₅ を選択(全てE8ルートであることを確認)。行列 Aᵢⱼ = 2⟨αᵢ,αⱼ⟩/⟨αⱼ,αⱼ⟩ を計算し、D6型標準Cartan行列と成分ごとに比較。**厳密一致。**

4. **閉包性(E8内でのルート系公理)の検証。**
   - 6つの単純ルートの整数結合の張る格子とE8ルート集合の交わりが、60本のD6集合と厳密に一致する — すなわちD6は(単に抽象的な部分系ではなく)閉部分系である。
   - すべての α, β ∈ D6 について、α+β ∈ E8 ならば α+β ∈ D6(全3600ペアで違反0件)。
   - D6は自身の全ルートに関する鏡映で不変(Weyl群閉包)。

## 検証の範囲と限界

- 本検証が確認したのは、ルート系レベルでの包含 D6 ⊂ E8 という**数学的事実**である(リー代数の言葉では、so(12) の e8 への正則部分代数としての包含に対応)。これはリー理論における既知の結果の、独立かつ再現可能な機械的再確認であり、分類 [A] の根拠として引用可能である。
- この包含に付随する**物理的解釈の正しさは本検証の対象外**である。物理的主張は、実験・観測との照合がなされるまで、それぞれの分類([B] 理論命題 / [C] 作業仮説)のままである。
- 今後の課題:同一命題の Lean 4 / Mathlib(`Mathlib.LinearAlgebra.RootSystem`)による形式化。これにより「機械検証」から「機械検査済み証明」への格上げが可能となる。
