## Abstract / 概要

本ノートは、v1.1〜v1.4で確立した幾何学的事実群を土台に、標準模型（Standard Model, SM）との構造的対応を5つの独立した観点から検証する。

(1) TTT基本方程式 $E = (xX+yY+zZ)\otimes\vec\omega + rR + iI + jJ$ の未定項「??」は、Cl(3)代数の外側に位置する「囲む演算子」——すなわちダークマターとしてのOOπ——として同定される。(2) 27点格子の12辺中点が「3方向×4個」に分解され、SMの12フェルミオンの「3世代×4種」構造と一致する。(3) フェルミオン電荷 $Q$ が幾何座標の線形結合 $Q=\alpha y+\beta z+\gamma$ として厳密に表現可能で、$y=2I_3$（弱アイソスピン）が完全一致する。(4) レプトン（4点自己完結）とバリオン（3点外部依存）で、OOπの役割が「潜在」と「顕在」に逆転し、質量の主体が転換する。(5) 陽子の閉じ込めエネルギー実測から $E_{\text{conf}} = 4\hbar c/R$ という単純式が0.9%精度で成立し、係数「4」がテトラ頂点数と一致する。

これらを総合し、**「可視質量（QCD閉じ込めエネルギー）と不可視質量（ダークマター）は同一のOOπの顕在態と潜在態である」**という統一仮説を[C]として提示する。

## Section A — 機械検証済み事実 [A]

### A-1. 27点格子の3×4分解（v1.5新結果）

3×3×3=27点格子の12辺中点は、3方向（x=0型, y=0型, z=0型）×4個に厳密分解される。各方向グループの頂点ベクトル和は独立にゼロ：

$$\Sigma(x=0\text{型}) = \Sigma(y=0\text{型}) = \Sigma(z=0\text{型}) = (0,0,0)$$

これはSMの12フェルミオン（3世代×4種）の分類構造と数値・構造ともに一致する。

### A-2. 電荷の線形対応（v1.5新結果）

フェルミオン電荷 $Q \in \{+2/3, -1/3, -1, 0\}$ が、幾何4点 $(0,\pm1,\pm1)$ の座標線形結合として表現可能。代表解：

$$Q = \frac{1}{3}y + \frac{1}{2}z - \frac{1}{6}$$

24通り中8通りで成立。特に $y = 2I_3$（弱アイソスピン第3成分）は厳密対応（$y=\pm1 \leftrightarrow I_3=\pm1/2$）。

### A-3. 陽子質量収支と閉じ込めエネルギー（実測事実の再確認）

- 構成クォーク質量和: $2m_u + m_d = 8.99$ MeV
- 陽子実測質量: $938.272$ MeV
- 閉じ込め由来: $929.28$ MeV（陽子質量の**98.9%**）

### A-4. TTT-OOπ閉じ込め関係式の数値検証（v1.5核心）

$$E_{\text{conf}} = \frac{4\hbar c}{R_{OO\pi}}$$

陽子で検証：予言 $R_{OO\pi} = 0.8494$ fm、実測 $R_p = 0.8414$ fm、**一致度0.9%**。

幾何係数 $N$ を実測から逆算した値 $N=3.962$ は、他の候補（$R/r=3$, $3+\sqrt6$, $\pi$, 体積比2.753）と26%以上外れる中、テトラ頂点数 $N=4$ にのみ0.9%で一致する。

## Section B — 理論的命題 [B]

### B-1. 基本方程式の完成形

Cl(3)代数の8次元は $\{1, X, Y, Z, YZ, ZX, XY, XYZ\}$ の8基底で完結する。TTT基本方程式：

$$E = (xX+yY+zZ)\otimes(\omega_x,\omega_y,\omega_z) + rR + iI + jJ + \boxed{??}$$

における「??」は、代数内部に**もう1つ加算する項ではあり得ない**（席が埋まっているため）。総和ゼロの要請（双曲の0）と、統合球OOπが「27点の外側」に位置する構造（v1.2）を合わせると、「??」は加算項ではなく**枠外の拘束演算子**として同定される：

$$E = \Theta_{OO\pi}\Big[(xX+yY+zZ)\otimes\vec\omega + rR + iI + jJ\Big], \qquad \text{拘束}: \Sigma_{\text{全体}} = 0$$

これは以前定式化された存在スイッチ $\Theta(v)$ を、囲む演算子として拡張したものとして解釈できる。

### B-2. 閉じ方の二分類とOOπの顕在/潜在

|  | 4点テトラ（レプトン） | 3点三角形（バリオン） |
|---|---|---|
| 閉じ方 | 自己完結 | 外部依存 |
| 4点目 | 運動から自給 | 囲むものが供給 |
| 質量の主体 | 点（100%） | 囲むもの（99%） |
| OOπ | 潜在 | 顕在 |

「自己完結できる構造」ではOOπは潜在化し、「自己完結できない構造」ではOOπは顕在化して質量として観測される、という命題。

### B-3. 3の再登場——TTTの共通因数

これまでの検証で「3」が独立に3回現れた：
- v1.3: $(R/r)^2$ 系列の共通因数（$3^2, 3^1, 3(7-4\varphi)$）
- v1.5-A1: 12=3方向×4分解
- v1.5-A2/電荷対応の $Y_w$ 比 1:3（クォークが3色で閉じることに由来）

すべて「螺旋1回転で完成する最小閉構造＝三角形の頂点数3」を指す。「3と4の関係」がTTTの中核であることの複数の独立証拠。

## Section C — 統一仮説と作業命題 [C]

### C-1. 顕在態/潜在態としてのOOπ統一仮説

$$\text{可視質量}(\text{QCD閉じ込め}) + \text{不可視質量}(\text{ダークマター}) = \text{同一のOO}\pi\text{の2状態}$$

「囲むものは、囲まれる側が自己完結しているとき仕事をしていないから見えない」——ダークマターの不可視性への構造的説明候補。宇宙の質量の大部分が「囲むもの由来（バリオン質量の99% + ダークマター）」で占められる観測事実との定性的整合。

### C-2. 反証可能な予言

TTT-OOπ関係式 $R = 4\hbar c/E_{\text{conf}}$ から派生する未検証予言：
- $R(\Lambda) = 0.777$ fm
- $R(\Omega^-) = 0.566$ fm

これらの半径が将来精密測定されたとき、予言値から系統的にズレれば式は棄却される。

### C-3. 適用範囲の明示

TTT-OOπ関係式は**バリオン（3粒系）専用**である。パイ中間子（2粒系）で逆算すると $N \approx 0.44$ となり、どの幾何定数とも一致しない。これは弱点ではなく「テトラ構造（4）は3粒を閉じるための構造」という理論の内部整合性から予言される限定である。

### C-4. 未解決点

- 電荷対応の一意性: 24通り中8通りの割り当てが線形対応を許すが、幾何原理から一意に選ぶ方法は未確立

- 中間子系（2粒）でのOOπの記述は未整備
- v1.4で残った減衰定数 $C\approx0.243$ の閉じた形は依然未解決
