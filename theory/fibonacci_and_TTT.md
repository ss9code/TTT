## 1. フィボナッチ数列の数学的事実

### ✓ 定義

$$F_n = F_{n-1} + F_{n-2}$$

$$1,\ 1,\ 2,\ 3,\ 5,\ 8,\ 13,\ 21\ \ldots$$

### ✓ 黄金比への収束

$$\lim_{n \to \infty} \frac{F_n}{F_{n-1}} = \phi = \frac{1+\sqrt{5}}{2} \approx 1.618\ldots$$

| 比率 | 値 |
|------|----|
| $1/1$ | $1.000$ |
| $2/1$ | $2.000$ |
| $3/2$ | $1.500$ |
| $5/3$ | $1.667$ |
| $8/5$ | $1.600$ |
| $\vdots$ | $\vdots$ |
| $\infty$ | $1.618\ldots$ |

収束はするが**φに到達しない**。この「到達しない収束」は数学的事実である。

### ✓ 螺旋との関係

フィボナッチ数列は対数螺旋（黄金螺旋）と対応する。  
螺旋は閉じない——回転するたびに拡大し、同じ点には戻らない。

## 2. 正四面体螺旋との構造的対応

### ✓ 正四面体螺旋（テトラヘリックス）の事実

正四面体を面共有で連結すると螺旋を描く：

$$\text{各接続のずれ角} \approx 22.24°$$

一周に約16個が必要。完全には閉じない——**非整合螺旋**。

### ◎ フィボナッチ螺旋との共通構造

| 性質 | 正四面体螺旋 | フィボナッチ螺旋 |
|------|------------|----------------|
| 閉じない | ✓ | ✓ |
| 到達しない収束 | ✓（非整合） | ✓（φへ漸近） |
| 自己相似性 | 部分的 | 強い |

**共通する本質：どちらも「完全には閉じない」「完全な値には到達しない」。**

これは数学的同一性ではなく、**構造的アナロジー**である。

### ◎ √3から√5への移行

```
√3：正四面体・正八面体・立方体（閉じた安定構造）
　　平面安定多角形（3・4・6角形）

√5：正五角形・正二十面体（球面で安定）
　　フィボナッチ・黄金比（閉じない収束）
```

$$\sqrt{3} \xrightarrow{\text{螺旋・閉じない}} \sqrt{5}$$

√3の構造が螺旋を描くことで√5の構造へ移行する——  
これは**構造的対応として有効**だが、数学的に厳密な導出はまだない。

$$\phi^2 = \phi + 1 \quad \Longleftrightarrow \quad F_n = F_{n-1} + F_{n-2}$$
