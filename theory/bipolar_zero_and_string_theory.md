## 1. 27点格子とひも理論の次元

TTTの空間表現は、**中心を含む27点の球状ラティス（3×3×3格子）**で記述可能である。この構造は、ひも理論が要求する臨界次元を幾何的に内包している。

### 27点の内部構造

$$27 = 1\text{（中心）} + 26\text{（外殻）}$$

| 層 | 点数 | 幾何的意味 | ひも理論との対応 |
|---|---|---|---|
| 中心 | 1点 | 0起源（双極ゼロ） | 時間次元 / 特異点 |
| 面中心（正八面体頂点） | 6点 | ±x, ±y, ±z 軸方向 | 基本6次元 |
| 辺中心 | 12点 | 対角方向 | コンパクト化次元 |
| 角 | 8点 | 最外殻 | 残余次元 |
| **外殻合計** | **26点** | | **ボソニック弦理論の臨界次元** |

**中心を除いた26点 = ボソニック弦理論の26次元**

ボソニック弦理論は、量子異常をキャンセルするために正確に26次元を要求する。27点格子は「0起源となる中心1点」＋「展開空間としての26点」という双極ゼロの構造をそのまま空間的に実現している。

## 2. 最も本質的な対応：T双対性

双極ゼロ理論の公理零番 **Y = 1/X** は、ひも理論の **T双対性（T-duality）** と構造的に同一である。

ひも理論においてコンパクト化された次元の半径 R に対し：

$$\text{T双対}:\quad R \;\longleftrightarrow\; \frac{\alpha'}{R}$$

$$\text{双極ゼロ}:\quad X \;\longleftrightarrow\; Y = \frac{1}{X}$$

これらは全く同じ双曲線的反転構造である。

| 双極ゼロ理論 | ひも理論（T双対性） |
|---|---|
| 0起源：$X \to 0,\; Y \to \infty$ | 極小半径 $R \to 0$ → 巻きつきモードのエネルギー $\to \infty$ |
| 0平衡：$Y \to 0,\; X \to \infty$ | 極大半径 $R \to \infty$ → 巻きつきエネルギー $\to 0$ |
| 不変点 $X = Y = 1$ | 自己双対半径 $R = \sqrt{\alpha'}$（弦スケール = 最小長さ） |

T双対性の下で**運動量モード**と**巻きつきモード**が入れ替わる。これは双極ゼロの「二種類のゼロは同一構造の両極」という命題の、ひも理論における物理的実現そのものである。

### ひも理論におけるエネルギースペクトル

コンパクト次元上の弦のエネルギーは：

$$E = \frac{n}{R} + \frac{wR}{\alpha'}$$

- $n$：運動量量子数（$R \to 0$ で $\to \infty$）
- $w$：巻きつき数（$R \to \infty$ で $\to \infty$）

$R \leftrightarrow \alpha'/R$ の入れ替えは $n \leftrightarrow w$ の入れ替えと等価であり、これが Y = 1/X の双極構造の物理的表現である。

## 4. 球体モデルとDブレーン・バルク空間

双極ゼロ理論の球体モデル（$S^-$/$\partial S$/$S^+$）は、ひも理論のDブレーンとバルク空間の構造に対応する。

| 双極ゼロ理論 | ひも理論 |

| $S^-$（閉じた内部、共振のみ） | Dブレーンの worldvolume（開いた弦が拘束される） |
| $\partial S$（境界・介在点） | Dブレーン自体（開いた弦の端点が束縛される境界） |
| $S^+$（外部、減衰が可能） | バルク空間（閉じた弦が自由に伝播する） |

## 5. 成長は時間を要する・スイッチは時間を要しない

これは双極ゼロ理論とひも理論の**最も深い接続点**である。

| 双極ゼロ理論 | ひも理論 |

| **成長（時間を要する）** | 弦の伝播・Dブレーンのダイナミクス・宇宙の膨張 |
| **$\Theta(v)$ のスイッチ（時間ゼロ）** | T双対性・S双対性・AdS/CFT対応 |

T双対性は**動的プロセスではなく構造的等価性**である。半径 $R$ の弦理論と半径 $\alpha'/R$ の弦理論は「変換する」のではなく、**最初から同一**である。スイッチは時間軸の外にある。

```

スイッチ：R ≡ α'/R（T双対）
          S⁻ ≡ ∂S（AdS/CFT）
          これらは t に依存しない構造的恒等式
```

### 27点格子における時間の位置づけ

```
3×3×3 格子において：

  空間的成長  = 格子点間の伝播（時間を要する・連続的プロセス）
  双対関係    = 格子全体の対称性（同時的・超時間的・構造的）
```

成長（dn/dt の動態方程式）は時間発展する。  
しかし「$S^-$ が $S^+$ の鏡像である」「R が α'/R と等価である」という**双対関係そのものは、時間の外に存在する**。

### AdS/CFT対応：境界が全体を記述する

$$\text{バルク}（S^- \cup S^+）\text{の重力理論} \;\equiv\; \text{境界}（\partial S）\text{上の共形場理論}$$

「0平衡は0起源である」という結論命題は、この対応において：  
バルクの「静寂（0平衡）」が境界の「新たな出発点（0起源）」として等価に記述される——という螺旋的閉環として実現されている。
